The backend follows a typical layered Spring Boot structure\+:


\begin{DoxyItemize}
\item {\ttfamily controllers/} expose REST endpoints under {\ttfamily /api/\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}}
\item {\ttfamily services/} contain business logic
\item {\ttfamily repositories/} contain persistence access (Spring Data)

\begin{DoxyItemize}
\item All endpoints under {\ttfamily /api/\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}} require authentication by default.
\item Public endpoints\+:
\begin{DoxyItemize}
\item {\ttfamily POST /api/auth/login}
\item {\ttfamily POST /api/auth/register}
\item {\ttfamily POST /api/users/register}
\end{DoxyItemize}
\item Authentication is configured as {\bfseries{stateless}}. Unauthorized requests return {\ttfamily 401}.
\item The project currently uses {\bfseries{HTTP Basic}} for protected endpoints.
\end{DoxyItemize}
